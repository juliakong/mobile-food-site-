First Page has an about us blurbs, but has glowing down arrow to prompt user to scroll downwards. They could also click the arrow for an automatic scroll to the next page down

Second page shows options for navigation. The order now button is outlined for emphasis

Upon clicking the Order Now button, the third page shows up, displaying the top 5 entrees to order from. These are presented as images with the name below and an "Add" botton below each option to add this dish to the cart. At the bottom right hand corner, there is a button to click to place the order, as well as a cart total that is updated after each order addition. Please ignore the side text outside of the image.

When the image is pressed on by the user, a window pops up beside it that has a larger image, description, and price. The background dims.

When the Order button is selected, a new page shows up that allows the user to select the delivery time from a drop down menu. They can also confirm their contact information drawn from Facebook. If correct, they will press "Place Order"

The last page thanks them for their order and lets them know a confirmation text will be sent. They can choose to return home with the last button
